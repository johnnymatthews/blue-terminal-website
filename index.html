Thanks for taking a look at this template! Everything's been written in plain-old HTML and CSS, there are no fancy frameworks here. There isn't any JavaScript. It's all been created from the ground up to keep things fast and easy to customize. The best place to start changing things is in the style.css file. There's a bunch of variables at the top of the file for things like colour and font-size.

This template was created by me, Johnny. I've also used the incredible int10h font by Int10H, and Meyerweb's Reset.css. If you would rather not include these extras in your project, remove the top 9 lines from style.css. Keep in mind that this will alter how this site looks.

If you've got any questions or have any suggestions for this template, raise an issue or make a pull request over on GitHub.
