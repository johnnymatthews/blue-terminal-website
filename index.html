Thanks for taking a look at this template! Everything's been written in plain-old HTML and CSS, there's no fancy frameworks here. There isn't even any JavaScript. It's all been created from the ground up to keep this fast and easy to customize. The best place to start changing things is probably the style.css file. There's a bunch of CSS variables at the top of the file for things like color and font-size.

This template was created by me, Mohnjatthews. I've also used Reset.css by Meyer Web and the int10h font by Int10H.

If you've got any questions or have any suggestions for this template, make raise an issue or a pull request over on GitHub.
